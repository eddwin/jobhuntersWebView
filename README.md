====================
JOBHUNTERS
---------------------

## Usage

This elemental WebService consumes the gem jobhunters to obtain information of some job offers in CA.

## Handles:

>GET /api/v1/
>returns JSON of job offers info: category (category), job_offers(Array of joboffers)
>POST /api/v1/
